# Python-Portfolio
This is a portfolio of my coding from AP Comp Sci Principles 2025-26.

**adventure.py** - This program is a choose-your-own-adventure that takes the user through a story and lets them make choices that influence the outcome.

**bottles.py**- This program prints the lyrics to the song 99 Bottles of Milk on the Wall

**calculator.py** - This program is a simple calculator where the user inputs numbers and an operation type (+-x/) and the program prints the answer.

**dogbreed.py** - This program lets users look up specific information about dog breeds or search for dogs based on their traits.

**fortunes.py** - This program takes input from the user about what kind of fortune they want and then prints their fortune and lucky numbers.

**grade.py** - This program takes a score input and prints the letter grade it should earn.

**grocery.py** - This program lets users see the price of grocery items and determine what they can buy, including with a discount applied.

**hogwarts.py** - This program takes a name input from the user and then prints the Hogwart House they belong to, after some suspense.

**madlibs.py** - This program takes word inputs from the user and then puts them into a MadLibs type format to create a funny story.

**nickname.py** - This program takes input from the user about their personality and uses it to determine what dessert they are.

**pokemon.py** - This program allows a user to train, rest, or battle with their Pokemon, and earn points so the Pokemon evolves and can beat the final boss.

**todo.py and todolist.txt** - This program is a to-do list where you can add and remove items and mark them as done. The .txt goes with to store the lists.

**weather.py** - This program takes input from the user about the weather in degrees Farenheit and gives a recommendation on what to wear.
